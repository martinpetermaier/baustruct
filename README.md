# Baustruct 🏗️

> BauGPT's Procure-to-Pay platform — inspired by Comstruct, built for the BauGPT ecosystem.

## What is Baustruct?

Baustruct automates the entire material procurement cycle for construction companies:

```
Order → Digital Delivery Note → AI Verification → Booked Invoice
```

**3 User Personas:**
- 🏗️ **Baustelle** (Field) — delivery calendar, digital delivery notes, mobile app
- 🛒 **Einkauf** (Procurement) — product catalog, contract prices, order management
- 💼 **Finanzen** (Finance) — automated invoice verification, ERP booking, AI coding suggestions
